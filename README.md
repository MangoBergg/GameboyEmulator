# Gameboy Emulator in Rust as a side project.
My goal is to recreate the behavior of the original Nintendo Gameboy - Which has probably been done a thousand times before. 

## Reference
 - The main bible is 'Pan Docs' for the hardware reference.

## Validation
 - Correctness will be verified using community test ROMs: 'https://github.com/c-sp/game-boy-test-roms'

## Why?
To play pokémon on my own emulator.
